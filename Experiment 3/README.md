# Experiment 3: 
Experiment Title:
File and Directory Management and File Permissions in Linux

## Aim
To study and practice Linux commands for file and directory management, understand Linux file and directory permissions, and create and edit files using text editors such as nano and vi.

## Theory
Linux follows a hierarchical file system starting from the root directory /. Files store data, directories store files/subdirectories. Permissions (rwx) apply to Owner, Group, Others.

## Commands Used
| Command | Description |
|---------|-------------|
| pwd     | Display current directory |
| ls / ls -l | List files & details |
| cd      | Change directory |
| mkdir   | Create directory |
| touch   | Create empty file |
| cat     | Display file contents |
| nano    | Edit with nano |
| vi      | Edit with vi |
| chmod   | Change permissions |
| stat    | Display file status |

## Step-by-Step Procedure
1. Login and open Terminal.
2. `pwd`
3. `ls` and `ls -l`
4. `mkdir linux_lab && cd linux_lab`
5. `touch file1.txt file2.txt`
6. `nano file1.txt` (type text → Ctrl+O → Enter → Ctrl+X)
7. `vi file2.txt` (i → type → Esc → :wq)
8. `cat file1.txt`
9. Change permissions & check with `chmod` and `stat`.
